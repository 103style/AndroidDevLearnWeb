css过渡效果 transition

鼠标移动到我身上，看看 transition效果,

语法 transition: 要过渡的属性（all 表示全部） 用时(s) 运动曲线（过滤速度 ease/ease-in/ease-out/ease-in-out ） 何时开始(触发的演示);

示例 transition: all 2s ease-in-out;
